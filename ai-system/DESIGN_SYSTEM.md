# SYT Design System

**Production-ready design system for Task & Habit Tracker (Telegram Mini App)**

A comprehensive, dark-theme-first design system built with React, Tailwind CSS v4, and Radix UI primitives.

## 📦 Design Tokens Extended

### New Elevation Levels
- surface-1, surface-2, surface-3 for layered interfaces
- overlay for modal backgrounds

### New Divider System
- divider-subtle, divider, divider-strong for content separation

### Overlays & Backdrops
- backdrop and backdrop-blur for modals and bottom sheets

### Focus & Disabled States
- focus-ring for accessibility
- disabled-bg, disabled-border, disabled-text for non-interactive states

## 🧩 New Components

### Navigation & Structure
- Modal (desktop)
- BottomSheet (mobile)
- Tabs (default, pills, underline variants)
- SegmentedControl
- FloatingActionButton (FAB)

### Task-Specific
- TaskListItem (compact, expandable, draggable)
- SwipeActions
- CalendarStrip
- Progress indicators (bar, circular, streak, completion stats)

### UI Feedback
- Toast/Snackbar
- Alert (inline)
- ConfirmDialog
- Divider (with label support)

### Utilities
- DragHandle
- Icon System (organized by category)

## 📱 New Screens

1. **Task Details** - Full task view with actions and activity
2. **Create Task** - Form with all task options
3. **Statistics** - Analytics dashboard with charts and achievements

## 🎨 Icon System

Organized icons from lucide-react:
- navigation, actions, time, status, priority
- taskManagement, progress, ui

## 🚀 All Components Production-Ready

- Full TypeScript support
- Accessible (ARIA labels, keyboard navigation)
- Responsive (mobile-first)
- Consistent styling
- Dark theme optimized
- Light theme tokens prepared
